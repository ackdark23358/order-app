# 커피 주문 앱 - 프런트엔드

## 개발 환경 설정

### 필수 요구사항
- Node.js (v18 이상)
- npm 또는 yarn

### 설치 방법

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 프로덕션 빌드:
```bash
npm run build
```

4. 빌드 미리보기:
```bash
npm run preview
```

## 기술 스택
- React 18
- Vite 5
- JavaScript (ES6+)
- Tailwind CSS 3.4
